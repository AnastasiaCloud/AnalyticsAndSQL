#Агентство недвижимости

#О проекте
Проект выполнен в рамках учебного курса "Разработка и защита баз данных".
Разработана структура БД для агентства недвижимости. Также созданы таблицы, связи и хранимые процедуры с обработкой ошибок.

#Инструментарий
DBeaver - создание структуры БД.
SQL Manager Lite for MySQL - создание и заполнение таблиц, просмотр общей структуры и результата хранимых процедур.
MySQL - создание хранимых процедур с обработкой ошибок.
DataLens - создание дашборда с аналитикой данных.

#Структура БД
ER-Diagram (images/Diagram.png)

#Хранимые процедуры
1. Поиск квартир по типу, заданному пользователем.
2. Поиск информации о квартире по адресу.
3. Обработка ошибок - если данные не найдены, возвращает всю таблицу.

#Дашборд
Выполнен при помощи BI-системы Yandex DataLens (images/Dashboard_datalens_analytics.png)

Ключевые выводы:
- Пик продаж — середина марта.
- Наибольший спрос — 2-комнатные квартиры (40%).

Рекомендации:
- Расширить предложение 2-комнатных квартир.
- Скорректировать рекламный бюджет на пиковые периоды.
