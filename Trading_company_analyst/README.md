#Торговое предприятие

#О проекте
Проект выполнен в рамках учебного курса "Разработка и защита баз данных", учебный проект с выполнением анализа данных, полученных из базы данных MySQL. Проведен анализ по остаткам товаров, маржинальности, спросу и рейтингу производительности товаров.

#Инструментарий
DBeaver/DBDesigner - создание структуры БД.
SQL Manager Lite for MySQL - создание и заполнение таблиц, просмотр общей структуры и результата хранимых процедур.
MySQL - создание хзапросов для создания анализа.
DataLens - создание дашборда с аналитикой данных.

#Дашборд
Выполнен при помощи BI-системы Yandex DataLens (images/Dashboard.png)

Ключевые выводы:
- Спросом пользуются 76% товаров.
- 8 позиций товаров заканчиваются на складах.
- 2 товара имеют большую разницу в себестоимости и цене - точки роста.

Рекомендации:
- Провести инвентаризацию на складах в целях избежания потери выручки.
- Проверить сезонность, качество и цену товаров на позиции, не пользующиеся спросом (24% от общего количества).
- Усилить продвижение товаров, имеющих высокую маржинальность.
